    text2svg.TextInfo¶

    class text2svg.TextInfo(text: unicode, filename: unicode, width: int, height: int, font_size: int = 10, font_style: text2svg.ctext2svg.Style = <Style.NORMAL: 0>, font_weight: text2svg.ctext2svg.Weight = <Weight.NORMAL: 400>, font_variant: text2svg.ctext2svg.Variant = <Variant.NORMAL: 0>, font: unicode = 'Sans', START_X: float = 0.0, START_Y: float = 0.0, text_setting=None)¶

    Bases: object

    This is the class which validates the arguments passed. This must be passed to text2svg().

    Parameters

    • text (str) – Text to convert.

    • filename (str) – The filename to write into.

    • width (int) – The width of the SVG.

    • height (int) – The height of the SVG.

    • font_size (int) – Size of font to write.

    • font_style (Style) – The style of font. Should be one from the defined enum.

    • font_weight (Weight) – The Weight of Font. Should be one from the defined enums.

    • font_variant (Variant) – An enumeration specifying capitalization variant of the font. Should be from the one defined.

    • font (str) –

      The font which the text must use.

      Important

      This font must be installed in the system or registered using register_font()

    • START_X (float) – Where to move the Cairo.Context before writing? Specify the x-coordinate here. Also, please see cairo docs about where the pointer starts.

    • START_Y (float) – Where to move the Cairo.Context before writing? Specify the y-coordinate here. Also, please see cairo docs about where the pointer starts.

    • text_setting (CharSettings) – These attributes which allow individual colouring, font changing or changing style of the rendered SVG image.

    Returns

    The generated TextInfo with the specified attributes and modifications.

    Return type

    TextInfo

    Examples

    >>> TextInfo("Hello World","hello.svg",50,50)
TextInfo(b'Hello World')
    Raises

    • AssertionError – If the file name doesn’t ends with .svg.

    • ValueError – If the directory you are trying to create the file doesn’t exists

    Warns

    • Filename exits already.

    • Width set to zero.

    • Height set to zero.
